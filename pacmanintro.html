Pac-Man Tutorial Introduction

In this Pac-Man tutorial you will learn how to code and build a game using JavaScript and the Box2D game engine library.

You are provided with ten tutorials with varying levels of tasks and challenges. Each of the tutorials will build your JavaScript coding knowledge.

After some of the tasks, you will be asked to execute your code so that you can view what you have developed.

Code executed within the editor will also save into the index.js file. This content of this file can be uploaded to the editor by clicking on the read in code button.

Please enjoy this tutorial and use the code sharing and communication functions when an available lecturer is online.
